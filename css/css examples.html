Selectors:

selectors are used to select HTML elements so that we can apply styles to them.

types of selectors:
  • class selector: it represent the .(dot) in css file and in html file by using class name
  • Id selectors: it represent the #(hash) in css file/li>
  • elements selectors: selects the all elements of a given type
  • universal selectors: select all elements by using *(star) in css.
  • first child selectors: select first element by using :(colon) in css.
  • last child selectors: select last element by using :(colon) in css.
  • nth child selectors: selects elements according to their position inside a parent, starting from 1.
  • attribute selectors: Select elements with a specific attribute. or select html elements based on their attributes or attribute values.
  • pseudo selectors : 1.pseudo-class selector(:hover) 2.pseudo-element selector(::before)

    • main pseudo-class selectors: a:hover, button:active, input:focus, input:disabled, first-child, nth-child, a:visited, a:link

  • descendant selectors: A descendant selector is used to select elements that are inside another element (at any level of nesting).
  • group selectors: select all ements

Box model:

Box model is used to how much space an elemnt occupies on the page.

types of Box Model:
1. content box
2. border box

components of box model:

  1. content: actual text or image
  2. padding: space between content and border (or) space inside the border
  3. border: line arround the padding
  4. margin: space outside the border

Positioning & Layout:

it is used to control the location of HTML elements on a web page. (or) it decides where an element shuold appear on the screen.

types of positions:
  • static:-> default position
    ->we can apply any position it did not work is called static.
    ->top , bottom, left, right do not work.
  • relative:->there is no change content behaviour
    ->it takes current position as the reference.
    ->we can aply top, right, bottom, left.
  • absolute:->it takes only content width not accupy the space
    it always take parent element as the reference.
    ->we can apply top ,right,left,bottom
  • fixed:->we can apply top, right,left,bottom
    ->it takes only the content width not occupy the space.
    it takes viewport as a reference.
  • sticky:->we can apply top,right,left,bottom.
    no change the content width.
    ->occupy the space.
